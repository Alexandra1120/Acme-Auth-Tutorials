# ACME Tutorials

Welcome to the ACME AUTH Tutorials site! This collection provides hands-on tutorials for multi-messenger astronomy and related topics, provided by the [Gravitational Waves group at the Laboratory of Astronomy, Aristotle University of Thessaloniki (AUTH)](https://www.acme-astro.eu/auth/).


## Contents

- **[AresGW](aresgw/intro.md)**: Deep-learning searches for gravitational waves in LIGO-Virgo-KAGRA data.
- **[GWG](gwg/intro.md)**: Confusion-noise estimates for compact Galactic binaries observed by LISA.
- **[RNS](rns/intro.md)**: Equilibrium models of rapidly rotating relativistic stars.
- **[Eryn](eryn/intro.md)**: Parameter estimation for a galactic binary source with gravitational-wave and electromagnetic data.
- **[Universal Relations](universal_relations/intro.md)**: Machine-learning universal relations for rotating neutron stars and neutron-star surfaces.


## Getting Started

Browse the tutorials using the navigation menu on the left, or click on any of the topics above to jump directly to a specific tutorial.

## Interactive Notebooks

All notebooks in this site can be run interactively using Binder. Click the 🚀 rocket icon at the top of any page and select “Binder” to launch an interactive session.
