# Deep Universal Relations

This tutorial set demonstrates neural-network models for neutron-star universal relations, including examples for normalized moments of inertia, quadrupole moments, spin octupoles, circumferential radii, and related quantities.

Source repository: [gregoryPapi/deep-universal-relations](https://github.com/gregoryPapi/deep-universal-relations)

Reference: [arXiv:2508.05850](https://arxiv.org/abs/2508.05850)
