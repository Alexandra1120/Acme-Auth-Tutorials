## Welcome to ACME Tutorials on Binder!

This interactive environment allows you to run and modify the notebooks directly in your browser.

### Available Tutorials

1. **AresGW** (`aresgw/`)
   - Inspecting generated noise, injection, foreground, and background files
   - Evaluating trigger statistics from foreground/background event files
   - Inspecting AresGW model 1 triggers for O3a month 2

2. **GWG** (`gwg/gwg_tutorial.ipynb`)
   - Estimating LISA Galactic-binary confusion noise
   - Generating demo catalogues and TDI A, E, T data
   - Running the iterative confusion-noise reduction loop

3. **RNS** (`rns/RNS_Tutorial.ipynb`)
   - Building and driving the RNS C executable from Python
   - Computing rapidly rotating relativistic-star models and sequences
   - Exploring EOS-dependent neutron-star equilibrium properties

4. **Eryn parameter estimation** (`eryn/parameter_estimation_eryn_ucb_gw.ipynb`)
   - Simulating a galactic binary gravitational-wave signal
   - Modeling the electromagnetic light curve
   - Running Eryn for GW-only, EM-only, and joint EM-GW inference

5. **Universal Relations** (`universal_relations/`)
   - Machine-learning universal relations for rapidly rotating neutron stars
   - Neural-network models for neutron-star surface quantities
   - Deep-learning demos for multipole and radius universal relations

### Quick Start

1. Navigate to `aresgw/`, `gwg/`, `rns/`, `eryn/`, or `universal_relations/` in the file browser.
2. Open the notebook you want to run.
3. Execute the notebook cells in order.

### Tips

- **Run cells**: Press `Shift + Enter`.
- **Save work**: Use `File > Download`; changes are not saved back to GitHub.
- **Restart kernel**: Use `Kernel > Restart` if something goes wrong.
- **Timeout**: Sessions expire after a period of inactivity.
